# Amazon Product Analysis Dashboard

## 📌 Project Overview

This project analyzes Amazon product data using Excel and Power BI to identify product trends, customer ratings, discount patterns, and category performance.

The objective of this project was to transform raw product data into meaningful business insights through data cleaning, analysis, visualization, and dashboard development.

---

## 🛠️ Tools Used

- Microsoft Excel
- Power BI
- DAX (Data Analysis Expressions)
- Pivot Tables
- Data Visualization

---

## 📊 Dataset Information

The dataset contains Amazon product information, including:

- Product ID
- Product Category
- Product Rating
- Rating Count
- Discount Percentage

A total of 1,352 products across 212 categories were analyzed.

---

## 📈 Dashboard Features

### KPI Cards
- Total Products
- Average Rating
- Average Discount
- Total Categories
- Average Review Count
- Most Represented Category

### Visualizations
- Top Categories by Product Count
- Top Categories by Average Rating
- Top Categories by Average Discount
- Discount vs Rating Scatter Plot

---

## 🔍 Key Insights

- Customer satisfaction is not solely driven by discounts.
- High discounts do not guarantee popularity.
- USB Cables are the most represented category.
- Some highly discounted categories receive lower ratings.
- Product count alone does not determine popularity.

---

## 💡 Recommendations

- Focus on high-rating, high-engagement categories.
- Review categories with high discounts but low ratings.
- Use discounts strategically rather than relying on them alone.
